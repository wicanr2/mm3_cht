# 我在2007的心得
這個方法 中英文版都適用

首先你要準備 GameMaster

可以動態修改記憶體的(因為我懶得看 save**.mm3 的檔案內容)

首先進入遊戲 你會發現你在泉頂鎮

X = 2 Y =5 的地方

用 GameMaster search 2
前進一格到 X = 3 Y = 5
search 3
在前進一格到 X = 4 Y = 5
search 4

你會得到一個位址
用 memory editor

你的所在位址 就是 X 的位址

他的格式是這樣 X Y LOCATION
假設你目前在泉頂鎮 X = 3 Y = 5的地方

記憶體內的內容會是 03 05 01

那要怎麼去秘密區呢 （記住秘密區也是有一堆怪物 所以修改的時候要順便修改 X Y)

由 X 開始修改

改成 01 0E 69 然後存檔 在讀檔

你會發現眼前一片漆黑 請用照明術

科科 恭喜老爺 你已經在秘密區了 請在財寶堆中走來走去

慢慢的挑選黑耀石裝備吧

然後要離開的時候呼叫巫師先生
